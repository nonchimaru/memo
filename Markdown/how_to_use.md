# Markdown の基本的な使い方

Markdown とは、プレーンテキスト形式で文書を記述するための軽量マークアップ言語。
Cmd + Shift + V（Mac）を押すことでプレビューを表示。

## テキストの強調

**太字** または **太字**
_斜体_ または _斜体_

## リスト

箇条書きリスト

- アイテム 1
- アイテム 2
  - サブアイテム

番号付きリスト

1. アイテム 1
2. アイテム 2
   1. サブアイテム
